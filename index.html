1. Introdução

A VoltCalc Pro ("Aplicativo", "nós", "nosso") respeita a privacidade de seus usuários ("você", "seu"). Esta Política de Privacidade explica como coletamos, usamos, divulgamos e protegemos suas informações quando você usa nosso aplicativo móvel VoltCalc Pro.

O VoltCalc Pro é um aplicativo profissional de cálculos elétricos destinado a engenheiros, eletricistas e profissionais da área elétrica.

2. Informações que Coletamos

2.1 Dados Coletados Localmente

O VoltCalc Pro coleta e armazena exclusivamente no seu dispositivo:

  • Cálculos de Lei de Ohm: Tensão (V), Corrente (A), Resistência (Ω), Potência (W)
  • Cálculos de Queda de Tensão: Distância (m), Corrente (A), Bitola (mm²), Material (Cobre/Alumínio), Modo (Monofásico/Trifásico), Queda de Tensão (V), Percentual (%), Tensão Final (V)
  • Dados do Profissional: Nome do responsável, nome do projeto
  • Histórico de Cálculos: Data, hora, tipo de cálculo
  • Preferências do Usuário: Tema (claro/escuro), idioma, configurações de exibição
  • Status Premium: Informações sobre compras no aplicativo
Importante: Nenhum desses dados é enviado para nossos servidores ou para terceiros sem sua autorização explícita.

3. Dados NÃO Coletados

O VoltCalc Pro NÃO coleta:

  • ❌ Localização GPS
  • ❌ Contatos ou agenda
  • ❌ Fotos ou vídeos do dispositivo
  • ❌ Dados de identificação pessoal (CPF, RG, CNH, etc.)
  • ❌ Informações de cartão de crédito (processadas apenas pelo Google Play)
  • ❌ Histórico de navegação
  • ❌ Dados de chamadas ou mensagens

4. Como Usamos as Informações

As informações coletadas são usadas exclusivamente para:

  • Funcionalidade do Aplicativo: Realizar cálculos elétricos conforme Lei de Ohm e NBR 5410
  • Persistência de Dados: Salvar seu histórico de cálculos localmente para acesso futuro
  • Melhorias: Entender como você usa o aplicativo para melhorar funcionalidades
  • Suporte: Responder a suas dúvidas e feedback via email

5. Armazenamento de Dados

Todos os dados são armazenados localmente no seu dispositivo. O VoltCalc Pro não sincroniza dados com servidores remotos sem sua autorização.

6. Segurança de Dados

Implementamos medidas técnicas para proteger seus dados:

  • ✓ Armazenamento local seguro no seu dispositivo
  • ✓ Criptografia do sistema operacional
  • ✓ Sem transmissão não autorizada
  • ✓ Controle de acesso exclusivo

7. Direitos do Usuário

Você tem direito a:

  • Acessar: Visualizar todos os seus dados
  • Corrigir: Editar ou atualizar suas informações
  • Deletar: Limpar seu histórico ou desinstalar o aplicativo
  • Portabilidade: Exportar seus cálculos

8. Dados de Crianças

O VoltCalc Pro é destinado a profissionais. Não coletamos dados de menores de 13 anos intencionalmente.

9. Conformidade Legal

O VoltCalc Pro está em conformidade com:

  • LGPD (Lei Geral de Proteção de Dados - Brasil)
  • GDPR (Regulamento Geral de Proteção de Dados - UE)
